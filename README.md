 # Python Developer Assignment

## Overview
This project is created as part of a Python Developer assignment.  
The goal of this assignment is to demonstrate basic Python skills, problem-solving ability, and understanding of real-world backend concepts such as data processing, REST APIs, OOPS, SQL, and debugging.

The code is written in a simple way.

--- README.md


------------
## Task 1: Employee Data Processing (CSV)

### Description
This task reads employee data from a CSV file and performs basic analysis.

### Features
- Read employee data from `employees.csv`
- Count total employees
- Calculate average salary by department
- Find department with highest average salary
- Find highest paid employee(s)
- Filter employees who joined in 2023
- Export 2023 employees to a new CSV file

-------
## Task 2: REST API Integration

### Description
This task demonstrates how to consume and interact with a public REST API using Python.

### API used - 
https://jsonplaceholder.typicode.com/posts


### Features
- Fetch all posts
- Count total posts
- Filter posts by userId = 1
- Find the longest post title
- Create a new post using POST request
